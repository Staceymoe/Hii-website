# Hii Explainer Film

Standalone web page for the 72-second Hii flagship film.

## Required files

- index.html
- hii-explainer.mp4
- hii-explainer-poster.jpg

The page attempts muted autoplay and always preserves a one-tap play-with-sound path for browser compatibility.

This is the low-resolution review animatic. Replace the MP4 and poster under the same filenames when the final master is ready.
